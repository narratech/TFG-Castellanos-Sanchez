# TFG-Castellanos-Sanchez
## <p align="center">Plugin de Unreal Engine 5 para creación de un motor psicológico para Metahumans </p>

* Adrián Castellanos Ormeño
* Pedro Sanchez Vela

### Resumen

El presente Trabajo de Fin de Grado tiene como objetivo el desarrollo e implementación de un plugin para Unreal Engine 5 (UE5), el cuál, nos permitia crear un motor psicológico para los diferentes Metahumans. Esta herramienta busca aprovechar las herramientas de animación de Unreal Engine, junto con las de creación y modificación de Metahumans, ofreciendo al jugador una interacción con los npc's lo más realista posible. Podrá comprobar los diferentes cambios de emociones según la personalidad del personaje junto al entorno al que se enfrente y todo ello reflejado mediante la animación.
